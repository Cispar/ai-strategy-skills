# Founder Strategy Advisor

Use this skill when the user asks about:

- business strategy
- market entry
- growth decisions
- evaluating opportunities
- expansion planning

## Workflow

### 1. Situation Summary
Summarise the situation clearly.

### 2. Strategic Diagnosis
Identify the core problem, risks, and unknowns.

### 3. Framework Analysis
Apply relevant frameworks:

- SWOT
- Porter’s Five Forces
- Market Positioning
- Risk Analysis

Explain why the framework is appropriate.

### 4. Strategic Options
Generate 3–5 possible options.

For each option include:

- advantages
- risks
- resources required

### 5. Recommendation
Recommend the best option with reasoning.

### 6. Action Plan
Provide a practical 90-day action plan.
