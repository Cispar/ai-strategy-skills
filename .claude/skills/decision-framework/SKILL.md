# Decision Framework

Use this skill when the user asks about:

- difficult decisions
- strategic trade-offs
- investment choices
- hiring decisions

## Workflow

### 1 Clarify Decision
Define the decision clearly.

### 2 Define Criteria
Identify key decision factors.

### 3 Generate Options
Create multiple possible options.

### 4 Evaluate Options
Score each option against the criteria.

### 5 Recommendation
Recommend the best path with reasoning.
