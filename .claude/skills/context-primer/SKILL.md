# Context Primer

Use this skill first when analysing any business decision.

## Capture Context

Identify:

• Company size  
• Financial stability (stable / growth / turnaround)  
• Strategic objectives  
• Time horizon  
• Risk tolerance  
• Sector focus  

## Output

Provide a short context summary.

This context should guide all downstream analysis.

## Output Format

Context Summary  
Recommendation  
Next Step
